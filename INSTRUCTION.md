# Quality Assurance Engineer Take Home Exam

## Instructions
Pre-Requisite:
1.	Install java on Windows: (Optional)
•	http://java.com/en/download/
•	Follow default installation instruction. 
2. Install Python: (Mandatory)
•	https://www.python.org/ftp/python/2.7.9/python-2.7.9.amd64.msi
•	Follow default installation instruction.
3. Install WxPython 2.8x: (Mandatory)
•	http://sourceforge.net/projects/wxpython/files/wxPython/2.8.12.1/wxPython2.8-win64-unicode-2.8.12.1-py27.exe/download
•	Follow default installation instructions.  
•	On the last screen of the installation wizard, keep selected options “Compile…” and “Create batch files…”
4. Install Python EXTENSIONS for Windows 64bit: (Mandatory)
•	http://sourceforge.net/projects/pywin32/files/pywin32/Build%20219/pywin32-219.win-amd64-py2.7.exe/download
•	Install. Default installation

5. Setup Environment Path for Python: ( Mandatory)
•	Right click on “Computer” > “Properties”.
•	Click on “Advanced System Settings” link.
•	Click on “Environment variables” button.  
•	Under “System variables” section, Edit the “Path” variable.  At the end of the line, add the installation path of the Python you installed in step #2.  If default installation is followed, this will be the path:  “;C:\Python27;C:\Python27\Scripts”.  (This is two entries.).
•	Save your changes before exiting the window.

6. Install Robot Framework: (Mandatory)
•	Run “Command prompt”
•	Execute “pip install robotframework”

7. Install IDE For Robot Framework
Eclipse Oxygen (Recommended)
•	
o	Download Eclipse Oxygen here: https://www.eclipse.org/downloads/download.php?file=/oomph/epp/oxygen/R2/eclipse-inst-win64.exe
o	Select Eclipse IDE for Java Developers
o	Install RED - Robot Editor Plugin https://github.com/nokia/RED



Robot Frameowk RIDE (Optional)
•	
o	https://github.com/robotframework/RIDE/releases/v1.5.2.1/
o	Click and download ==> robot framework-ride-1.5.2.1.win-amd64.exe
o	Install default installation
8. Install Selenium and Appium Librarys: ( Mandatory)
•	
o	Run “Command promt”
o	Execute “pip install  robotframework-selenium2library”
o	Execute “pip install  robotframework-Appiumlibrary”
9. Download Latest Browser Driver
Chrome WebDriver: https://sites.google.com/a/chromium.org/chromedriver/downloads 
•	Download and save it in C:\Python27\Scripts
IE WebDriver: http://selenium-release.storage.googleapis.com/index.html
•	Download and save it in C:\Python27\Scripts
FF WebDriver: https://github.com/mozilla/geckodriver/releases
•	Download and save it in C:\Python27\Scripts

10.Download Git:
https://git-scm.com/downloads

https://github.com/DigixGlobal/devquiz05.git
